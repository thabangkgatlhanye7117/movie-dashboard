## Project Overview
The Mini Movie Dashboard is a small interactive application that displays a dataset of movies, allows filtering by genre, and visualizes movie counts per genre using a dynamic bar chart. This project demonstrates handling structured data, real-time filtering, aggregation, and visualization â€” key skills for data-driven applications.

---

## Problem Solved
Users often want to explore datasets quickly and gain insights at a glance. This dashboard provides a simple interface to filter movies and visualize trends, simulating real-world data analysis and decision-making tools.

---

## Features
- Filter movies by genre using a dropdown menu  
- View a dynamic table of movies with title, year, genre, and rating  
- Interactive bar chart showing the number of movies per genre  
- Updates chart in real-time based on user-selected filters  

---

## Data Relevance
- Captures structured JSON dataset  
- Processes and filters data dynamically  
- Aggregates counts for visualization  
- Demonstrates foundational dashboard skills for data analysis and presentation
